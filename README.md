# curly-octo-spoon
react函数式组件+typescript + tailwindcss + useState


# 运行
pnpm install
pnpm run dev

# 上传
git status
git add .
git commit -a -m beizhi
git push


# 解释
react - router - dom
会根据当前的路由配置，渲染出对应的组件，从而实现页面的切换和导航功能。
可以一个简单的导航功能，包括首页和关于页面的切换。用户点击导航链接时，会根据不同的路径渲染对应的组件。

# 安装
npm install react-router-dom
